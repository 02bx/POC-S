结构  
---
以下含@的目录表示用户可控区域，用户可根据需要*增加|修改|调用*其中的内容

| 目录 | 说明 |
| :-----  |:-----|
| `POC-T.py` | 程序入口 |
| script     | @脚本库 |
| plugin       | @工具库 |
| data       | @资源库 |
| output     | 默认输出位置 |
| lib        | 项目代码 |
| doc        | 文档及版权声明 |
| thirdparty | 第三方库 |
| api        | 搜索引擎接口 |
